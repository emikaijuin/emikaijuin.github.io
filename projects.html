REMOTE LIGHT CONTROL

This project is an Arduino build using a NodeMCU board. An LED is connected to the arduino, which is connected to a web application hosted on Heroku using web sockets. From the web app, a user is able to pick a color and the selected color is broadcasted through the channel to the Arduino. The data is parsed and sent to the LED as an RGB value, updating the color emitted from the diode.

In this way, a user is able to control an LED light from anywhere in the world so long as the Arduino is able to connect to a Wi-Fi network. At this stage, only one channel exists to control a single Arduino. In the future, the project will be updated to accept multiple devices by assigning each a unique ID upon start-up and saving it to user groups. Users could then control groups of devices, e.g. Living Room Lights, thus turning the web app into an effective smart home hub.

IDENTICON GENERATOR

A program which generates identicons mimicking those used by GitHub as default avatars. A string input is converted to a datastream, which is then used to generate a contrast color and a grid layout. Selected squares on the grid are colored in, and the overall configuration is drawn to a canvas and saved as a PNG format using Erlang's graphic module EGD.

FUZZIFY

This project utilizes ChunkyPNG by Willem van Bergen, a library which reads and writes PNG files. Fuzzify allows users to load local images and specify a level to which to "fuzzify" the image. The program will write a folder to the user's machine and there save a fuzzified image. These images could be used to analyze color composition of photos or generate color palettes. As a separate feature, the program also calculates and saves an RGB histogram of the image.

Future features for this program could include calculating and applying Gaussian blurs over the pixel map to generate abstract art based upon a fuzzified copy of the original image.

ROBOT ARM

This is Robot Arm with 6 degrees of freedom controlled by joysticks. Input from the joystick describing the angle of tilt in the X- and Y-axes is mapped to coordinate the 6 servo motors operating the arm. Range of motion includes rotation and extension of shoulder, extension of elbow, rotation and extension of wrist, and closure of hand to grasp an object.
